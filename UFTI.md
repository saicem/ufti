# UFTI - 飞盘盘格评测系统

## 项目概述

UFTI (Ultimate Frisbee Type Indicator) 是一个飞盘运动的"盘格"评测系统，类似于 MBTI 人格测试，用于评估飞盘运动员的风格和特点。

## 技术要求

- 单页面网站，使用 Vue3 Composition API + TypeScript 实现
- 保证网站完整可交互
- 适配桌面端和移动端
- 设计风格：活泼有趣
- 首页，每个题目，评测结果分为多页展示，不要全部展示在一页上

## 功能需求

### 评测系统

- 每个题目涉及一个维度
- 不同选项得到不同的分数，越符合该维度，分数越高
- 选项分数范围：0-10
- 选项按分数排序
- 不同题目有不同的权重
- 维度评分计算公式：(维度中所有题目的[权重 * 题目中选项的得分]) / 维度中所有题目的权重总和) * 10

### 盘格匹配

- 根据维度评分进行匹配
- 匹配条件：
  - 维度评分在某值以上
  - 维度评分在某值以下
  - 多个盘格同时匹配则以第一个匹配的为准
- 条件越复杂越苛刻的盘格顺序越靠前，越简单越宽松的盘格顺序越靠后。

### 评测维度

| 维度 | 说明 |
|------|------|
| 跑动能力 | 对位通常是不想追别人，还是能跑到对位不想追 |
| 摆脱能力 | 防守人站前边就说没有没有，还是有具体思路和方法摆脱 |
| 接盘能力 | 考虑正面冲刺接盘能力。接非常规盘，包含hammer，刀盘的能力。能否跳接盘。能否单手接盘 |
| 持盘心态 | 不敢传盘，鲁莽传盘还是能冷静按情况传盘 |
| 长传能力 | 考虑射程，能否在防守下稳定出盘，出盘稳定性，能否掌控速度和悬浮时间 |
| 短传能力 | 考虑是否能正反手快速出盘，破break能力，过顶盘，dump，有无跨步习惯 |
| 防守能力 | 考虑是否知道如何防守，知道看人下菜，紧紧跟住对方，合理逼向，换防 |
| 意识 | 在被poach时知道如何处理，知道什么时候该下底，什么时候该上切，什么时候该救盘。不会给出盘就立刻贴上去救盘而是给cutter处理时间，等需要时再救盘 |

### 评测题目

- 每个题目都有多个选项，用户需要选择一个
- 每个题目都有一个分数，分数越高，则题目对该维度越重要
- 不同题目的选项数量可不一致
- 每个选项都有一个分数，分数越高，说明用户符合该维度
- 同一维度下不同题目从不同角度评估，例如：
  - 长传能力：1.射程 2.出盘稳定性 3.是否能掌控速度和悬浮时间
- 题目内容和选项内容要有趣。甚至具有一些夸张的选项，例如：
  - 题目：你防守对方通常的情况是？
  - 选项：放你3m，只有强者才配让我认真
  - 选项：通常能够紧紧粘住对方，对方需要一些技巧才能摆脱
  - 选项：能提供威胁，但对方仍然可以轻松摆脱
  - 选项：能追上就不错了

### 结果展示

- 显示人格名称
- 显示人格描述（可以有趣一些）
- 显示维度得分
- 示例：永动机 - 飞盘场上不知疲倦的跑动者

## 数据结构

- 将题目、选项、维度、盘格数据存储在单独的 JSON 文件中
- 网站加载时从 JSON 文件中读取数据

## 测试要求

- 创建 test.py 文件
- 用于测试是否存在某种情况匹配多个人格
- 方便进行数据调整

## 注意事项

- 使用飞盘相关词汇，避免使用与球相关的词汇
- 例如：使用"传盘"而非"传球"，使用"场地"而非"球场"
